> **Date:** 2026-01-24
> **Author:** Arturo Vargas Cuevas
> **↑ Parent:** [[1-development-guidelines]]

## Overview

| Element | Convention | Example |
|---|---|---|
| **Functions** | `snake_case` with module prefix | `hal_spi_transmit()` |
| **Variables (local)** | `snake_case` | `distance_mm` |
| **Variables (global)** | `g_` prefix + `snake_case` | `g_system_tick` |
| **Variables (static file-scope)** | `s_` prefix + `snake_case` | `s_tx_buffer` |
| **Pointer parameters** | `p_` prefix | `p_data`, `p_config` |
| **Constants / Macros** | `UPPER_SNAKE_CASE` | `DETECTION_THRESHOLD_MM` |
| **Types / Structs** | `snake_case` + `_t` suffix | `sensor_config_t` |
| **Enums (type)** | `snake_case` + `_t` suffix | `parking_state_t` |
| **Enums (values)** | `UPPER_SNAKE_CASE` with type prefix | `PARKING_STATE_OCCUPIED` |
| **File names** | `snake_case` | `hal_spi.c`, `hal_spi.h` |
| **ISRs** | Vendor-defined vector name | `SPI1_IRQHandler()` |
| **Register defines** | `UPPER_SNAKE_CASE`, peripheral prefix | `SPI1_CR1_SPE_BIT` |

## Functions

Use `module_action_object()` pattern:

```c
/* HAL layer */
hal_status_t hal_spi_transmit(uint8_t *p_data, uint16_t len);
hal_status_t hal_i2c_read(uint8_t addr, uint8_t *p_buf, uint16_t len);
hal_status_t hal_gpio_write(gpio_pin_t pin, gpio_state_t state);

/* Driver layer */
sensor_status_t tof_sensor_init(const sensor_config_t *p_config);
sensor_status_t tof_sensor_read_distance(uint16_t *p_distance_mm);
rf_status_t     rf_module_send_packet(const uint8_t *p_payload, uint8_t len);

/* Application layer */
void            parking_detect_update(void);
parking_state_t parking_detect_get_state(void);
cal_status_t    cal_send(const uint8_t *p_data, uint16_t len);
```

**Rules:**
- Public functions are prefixed with their module name.
- `static` (file-scoped) functions may omit the module prefix.
- Init/deinit functions: `module_init()`, `module_deinit()`.
- Boolean queries: `module_is_condition()` → `sensor_is_ready()`.

## Variables

```c
/* Local variable */
uint16_t distance_mm = 0U;
bool     is_occupied = false;

/* Global variable (avoid when possible) */
volatile uint32_t g_system_tick = 0U;

/* Static file-scope variable */
static uint8_t s_tx_buffer[MAX_PACKET_SIZE];
static bool    s_initialized = false;

/* Pointer parameter */
hal_status_t hal_spi_transmit(uint8_t *p_data, uint16_t len);
```

**Physical units in the name** when relevant:

```c
uint16_t distance_mm;       /* millimeters */
uint32_t timeout_ms;        /* milliseconds */
uint32_t frequency_hz;      /* hertz */
uint16_t voltage_mv;        /* millivolts */
uint32_t sleep_period_s;    /* seconds */
```

## Constants & Macros

```c
/* Module-prefixed constants */
#define HAL_SPI_MAX_TRANSFER_SIZE   (256U)
#define PARKING_THRESHOLD_MM        (200U)
#define CAL_MAX_RETRIES             (3U)
#define RF_CHANNEL                  (76U)

/* Boolean-style macros */
#define HAL_GPIO_PIN_SET            (1U)
#define HAL_GPIO_PIN_RESET          (0U)

/* Bit manipulation macros — document the register */
#define SPI1_CR1_SPE_BIT            (1U << 6U)   /* SPI Enable */
#define SPI1_CR1_MSTR_BIT           (1U << 2U)   /* Master mode */
```

## Types & Structs

```c
/* Struct type — descriptive, with _t suffix */
typedef struct
{
    uint8_t  i2c_addr;
    uint16_t timeout_ms;
    bool     use_interrupt;
} sensor_config_t;

/* Opaque handle type for driver instances */
typedef struct tof_sensor_handle tof_sensor_handle_t;
```

## Enums

```c
/* Enum type name: module + _category + _t */
typedef enum
{
    PARKING_STATE_UNKNOWN   = 0,
    PARKING_STATE_OCCUPIED  = 1,
    PARKING_STATE_UNOCCUPIED = 2,
} parking_state_t;

typedef enum
{
    HAL_STATUS_OK           = 0,
    HAL_STATUS_ERROR        = 1,
    HAL_STATUS_BUSY         = 2,
    HAL_STATUS_TIMEOUT      = 3,
} hal_status_t;
```

**Rules:**
- Enum values share a common prefix matching the type name (without `_t`).
- Always assign explicit values.
- Trailing comma on the last enumerator for cleaner diffs.

## File Names

```
hal_spi.c           hal_spi.h
hal_i2c.c           hal_i2c.h
tof_driver.c        tof_driver.h
rf_driver.c         rf_driver.h
parking_detect.c    parking_detect.h
cal.c               cal.h
error_codes.h
common_types.h
```

**Rules:**
- All lowercase, underscore-separated.
- Source/header pairs share the same base name.
- No generic names (`utils.c`, `helpers.c`) — be specific.

---

## ISRs (Interrupt Service Routines)

Use the exact vector name from the vendor startup file. Do not rename.

```c
/* MCU vendor CMSIS vector names — do not rename */
void SPI1_IRQHandler(void);
void I2C1_EV_IRQHandler(void);
void TIM2_IRQHandler(void);
void EXTI0_IRQHandler(void);
```

## Register Definitions

```c
/* Format: PERIPHERAL_REGISTER_FIELD */

#define ADC1_CR_ADSTART_BIT     (1U << 2U)
```
