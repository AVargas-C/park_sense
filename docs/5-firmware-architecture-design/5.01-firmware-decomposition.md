## 1. Purpose

This document defines the complete file inventory for ParkSense firmware: every source file, header, linker script, and test file; which build target each belongs to; and what API each module exposes and consumes. This is the authoritative reference for setting up `firmware/src/` before writing any code.

## 4. API Contracts

Each module exposes one public header. Layers above may only call the functions listed there.

| Module | Public header | Exposes | Consumes |
|--------|--------------|---------|----------|
| BSP | `bsp.h` | `BSP_Init()`, `BSP_EnterStop2()`, `BSP_GetTick_ms()`, `BSP_IWDG_Pet()` | STM32 HAL directly |
| BSP Secure | `bsp_secure.h` | `BSP_Secure_Init()` (configures SAU, GTZC, S-MPU) | STM32 HAL Secure registers |
| Sensor drivers | `sensor_api.h` | `SENSOR_ToF_Init()`, `SENSOR_ToF_Read()`, `SENSOR_Mag_Init()`, `SENSOR_Mag_Read()` | `bsp.h` (I2C HAL) |
| RF driver | `rf_api.h` | `RF_Init()`, `RF_JoinNetwork()`, `RF_Send()`, `RF_SetRxCallback()`, `RF_InitCoordinator()` | `bsp.h` (IPCC HAL) |
| Net driver | `net_api.h` | `NET_Init()`, `NET_Connect()`, `NET_Send()`, `NET_Recv()`, `NET_Disconnect()` | `bsp.h` (SPI HAL) |
| PDM | `pdm.h` | `PDM_Init()`, `PDM_Update()`, `PDM_GetOccupancy()` | `sensor_api.h` |
| CPM | `cpm.h` | `CPM_Init()`, `CPM_SendReport()`, `CPM_ReceiveReport()`, `CPM_ForwardToServer()`, `CPM_Server_Tick()` | `rf_api.h`, `net_api.h` |
| Bootloader | `btl.h` | `BTL_ValidateImage()`, `BTL_SelectSlot()`, `BTL_JumpToApp()` | `bsp.h` (flash HAL, PKA HAL) |
| OTA | `ota.h` | `OTA_Init()`, `OTA_WriteChunk()`, `OTA_Finalise()` | `bsp.h` (flash HAL), `cpm.h` |

## 5. Shared Header Definitions

### `include/common_types.h`

```c
typedef uint8_t   ps_node_id_t;     /* 8-bit node identifier (0x01вЂ“0xFE) */
typedef uint8_t   ps_space_id_t;    /* 8-bit parking space identifier */
typedef uint32_t  ps_timestamp_t;   /* Seconds since epoch */

typedef enum
{
    PS_TARGET_NODE       = 0x01U,
    PS_TARGET_GATEWAY    = 0x02U,
    PS_TARGET_BOOTLOADER = 0x03U,
} ps_target_id_t;

typedef enum
{
    OCCUPANCY_FREE     = 0x00U,
    OCCUPANCY_OCCUPIED = 0x01U,
    OCCUPANCY_ERROR    = 0xFFU,
} ps_occupancy_t;
```
